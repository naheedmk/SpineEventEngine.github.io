Major directions of Spine development.

In Next Release

  • Chromium 79

    Migrate to Chromium 79.

In Development

  • Cross-Platform Support

    Support Linux and Mac environments.

Planned

  • ARM

    Support the ARM devices such as Raspberry Pi 3 with Linux and .NET Core 3.0.

  • Printing API

    Allow programmatically printing a web page with the different print settings silently without displaying the Print Preview dialog.

  • In-process DevTools

    Add functionality that allows displaying Chromium DevTools without configuring the Remote Debugging Port.
